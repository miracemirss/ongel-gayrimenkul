# Hero Section Background Image

## Gereksinimler

Ana sayfa hero section için monokromatik (siyah-beyaz/gri tonlu) bir arka plan görseli eklenmelidir.

### Dosya Adı
`hero-luxury-estate-bw.jpg` veya `hero-luxury-estate-bw.webp`

### Özellikler
- **Format**: JPG veya WebP (WebP önerilir - daha küçük dosya boyutu)
- **Boyut**: Minimum 1920x1080px (Full HD), tercihen 3840x2160px (4K)
- **Stil**: Monokromatik (siyah-beyaz/gri tonlu filtre uygulanmış)
- **Konu**: Lüks emlak, modern mimari, İstanbul silüeti veya benzeri lüks emlak teması
- **Kalite**: Yüksek çözünürlük, profesyonel görsel

### Kullanım
Görsel `frontend/public/images/` klasörüne yerleştirilmelidir.

### Not
Görsel hero section'da `background-size: cover` ile kullanılacak ve üzerine %35 opaklıkta siyah overlay eklenecektir.

# Logo Dosyası

## Gereksinimler

Navbar'da kullanılacak logo dosyası bu klasöre eklenmelidir.

### Dosya Adı
`ongel-logo.png` veya `ongel-logo.svg`

### Özellikler
- **Format**: PNG (şeffaf arka plan önerilir) veya SVG
- **Boyut**: Minimum 300px genişlik (yüksek çözünürlük önerilir)
- **Stil**: Logo siyah arka plan üzerinde görünecek, bu yüzden açık renkli veya beyaz logo tercih edilir
- **Arka Plan**: Şeffaf (PNG) veya beyaz/açık renkli logo

### Kullanım
Logo `frontend/public/images/ongel-logo.png` olarak yerleştirilmelidir.

### Not
- Logo navbar'da scroll durumuna göre otomatik olarak görünür olacaktır
- Şeffaf arka planlı logo önerilir (hero section üzerinde daha iyi görünür)
- SVG formatı responsive için daha iyi sonuç verir
